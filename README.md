# JNAJack

Java bindings to JACK Audio Connection Kit (http://jackaudio.org/)

## Installation

Gradle (Groovy):

```groovy
implementation 'org.jaudiolibs:jnajack:1.4.0'
```

Gradle (Kotlin DSL):

```kotlin
implementation("org.jaudiolibs:jnajack:1.4.0")
```

Apache Maven: 
```xml
<dependency>
  <groupId>org.jaudiolibs</groupId>
  <artifactId>jnajack</artifactId>
  <version>1.4.0</version>
</dependency>
```
